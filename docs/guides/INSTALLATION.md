## Installation Steps

### Chrome / Edge / Brave

1. **Download the Extension**
   - Click the "Download Extension" button on your dashboard
   - Save `SmartHireX-Extension.zip` to your computer

2. **Extract the Files**
   - Right-click the downloaded zip file
   - Select "Extract All" or "Unzip"
   - Remember the folder location

3. **Open Extensions Page**
   - Open Chrome
   - Navigate to `chrome://extensions/`
   - Or click Menu (‚ãÆ) ‚Üí Extensions ‚Üí Manage Extensions

4. **Enable Developer Mode**
   - Toggle "Developer mode" switch (top right corner)

5. **Load the Extension**
   - Click "Load unpacked" button
   - Select the extracted SmartHireX-Extension folder
   - Click "Select Folder"

6. **Pin to Toolbar** (Recommended)
   - Click the puzzle icon in your toolbar
   - Find "SmartHireX - AI Form Filler"
   - Click the pin icon to keep it visible

## How to Use

1. **Login to SmartHireX**
   - Click the extension icon
   - Login with your SmartHireX account
   - Or click "Sync Session" if already logged in

2. **Navigate to a Job Application**
   - Go to any job posting
   - Click "Apply" to open the application form

3. **Fill with AI**

## Troubleshooting

### Extension Not Showing?
- Make sure you pinned it to the toolbar
- Look for the purple SmartHireX icon

### "Please log in" Message?
- Click "Sync Session" if you're logged into SmartHireX
- Or click "Login to SmartHireX" to sign in

### Forms Not Filling?
- Make sure you're logged in to the extension
- Refresh the page and try again
- Some sites use custom form components that may require manual filling

### Extension Not Working After Update?
- Go to `chrome://extensions/`
- Find SmartHireX extension
- Click the reload icon (‚Üª)
